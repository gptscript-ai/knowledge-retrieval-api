# knowledge-retrieval-api

Standalone Knowledge Retrieval API Server to be used with Rubra

## Supported Documents

Currently, the following file types are supported:

- `.pdf`
- `.txt`
- `.md`
- `.log`
- `.rtf`

## Development

- Dependency Management: `uv`
- Linting & Formatting: `ruff`
